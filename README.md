## Features

- **Reduction to Essence**: A strict black-and-white design system. No gradients. No shadows. Just content.
- **Open Source SaaS**: Fully transparent codebase, deployed as a premium service.
- **Editorial Typography**: High-impact serif typography (`Playfair Display`, `Source Serif 4`) combined with technical monospace details.
- **Interactive 3D Elements**: Smooth, physics-based card interactions using `framer-motion` and CSS transforms.
- **Secure Authentication**: Integrated with **Clerk** for seamless user management and security.
- **Smart Profiles**: Automatic avatar generation using **Facehash** for unique, deterministic user identities.
- **Password Strength Meter**: Custom-built, minimalist password strength visualization powered by `zxcvbn`.

## Tech Stack

Built with the latest technologies for speed and precision:

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Backend API**: Hono (via Next.js API Routes)
- **Auth**: Clerk
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Package Manager**: Bun
